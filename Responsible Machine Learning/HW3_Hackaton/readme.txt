This folder contains the code used for the Responsible Machine Learning 2022/2023 Hackathon.

Files :
- justice_train.csv : train set justice from ETHICS.
- justice_test.csv : test set justice from ETHICS.
- justice_augmented.csv : augmented set from ETHICS and GPT-3.5.
- RML_HKT_DataAugmentation.ipynb : notebook used to generated data augmentation.
- RML_HKT_GPT2_FineTune.ipynb : notebook used to test, fine-tune GPT-2 on justice.

Remark : please modify some variables to make the code run.
